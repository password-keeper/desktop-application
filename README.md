# What is Password Manager?
Desktop application written in Python by larei.

## Installation
Install Python 3 and libraries with `pip install -r requirements.txt`.

## Usage
Create an api from [here](https://github.com/password-keeper/password-manager-backend) and place link to **api.py**. Run this command `python3 main.py` or `python main.py` and ta-da ðŸŽ‰. Now you can use the application.
